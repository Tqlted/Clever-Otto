<h1>Clever Otto</h1>

  <p>Clever Otto is a memory-based grid game where your challenge is to test and improve your memory skills. In this game, you are presented with a grid of buttons, each representing a different element in a sequence. Your task is to remember and reproduce the sequence by clicking the buttons in the correct order.</p>

  <h2>How to Play</h2>
    <ol>
        <li><strong>Starting a Round:</strong> Each round begins with a sequence of buttons being highlighted in a random order.</li>
        <li><strong>Remember the Sequence:</strong> Pay close attention to the order in which the buttons are highlighted.</li>
        <li><strong>Reproduce the Sequence:</strong> After the sequence is displayed, it's your turn to reproduce it. Click the buttons in the same order they were highlighted.</li>
        <li><strong>Increasing Difficulty:</strong> With each successful round, the sequence becomes longer, challenging your memory further.</li>
        <li><strong>Scoring:</strong> Earn points for each correct sequence reproduced. Try to achieve the highest score possible!</li>
    </ol>
